# ⚡ Proyecto: Monitoreo de Corriente Eléctrica

Este repositorio contiene la solución a la **Práctica N° 1**, un programa en Python diseñado para el monitoreo de carga en líneas de distribución.

## 🛠️ Cómo funciona el código
1. **Entrada:** Solicita corriente nominal y medida.
2. **Validación:** No permite valores negativos.
3. **Cálculo:** Determina el % de carga mediante la fórmula de relación de corrientes.
4. **Clasificación:** Indica si el estado es Normal, Carga Alta, Alerta o Sobrecarga.
